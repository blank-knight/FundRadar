## 第二步：信号生成

### 评分过程

信号生成由 `signal_generator.py` 的 `generate_daily_signal()` 完成。

**V2 五维权重（设计值）:**
```
博主共识  25%   → 实际=0   (无数据)
新闻情绪  20%   → 实际=0.3439 (偏正面)
散户情绪  15%   → 实际=-0.1872 (偏空)
资金面    25%   → 实际=-0.8076 (大幅流出)
行业动能  15%   → 实际=None  (无数据)
```

**降级逻辑触发:**
- industry_momentum_score=None → 触发权重重分配，资金面从25%提升
- blogger_consensus_score=0 → 不是None（有值），不触发重分配，权重保持
- 实际有效维度: 3/5（新闻+散户+资金面）

**综合评分:**
```
score = 0*25% + 0.3439*20% + (-0.1872)*15% + (-0.8076)*25% + (None)*15%
     = 0 + 0.0688 - 0.0281 - 0.2019 + 0
     = -0.1612 (负值)
```

负值映射 → **sell（减仓）**，置信度70%。
